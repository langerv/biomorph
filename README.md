# biomorph
A little game to test ideas about metamorphosis and RPG

## Installation
Game
```
pip3 install arcade==2.1.3
python game.py
```

To run aptitude_unittest.py please install additional packages:
```
pip3 install numpy
pip3 install matplotlib
```
